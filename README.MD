# UNITY TOWER DEFENSE

This is a simple tower defense game built using the Unity game engine. The game features various types of towers, enemies, and levels to provide an engaging experience for players.
Builded for a university final project.

The objective is to prove that using design patterns can improve code maintainability and scalability in game development.
